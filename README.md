# RoastMaster 3000 🤖🔥
A Discord bot that roasts your friends based on their messages in #general.

## Setup
1. Create `data/config.json` and add your Discord and OpenAI keys.
2. Run:
   ```bash
   pip install -r requirements.txt
   python main.py
3. Invite your bot and type !roast @username in #general.
